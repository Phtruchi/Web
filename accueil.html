Body initial

Le travail à réaliser - Etude de cas : Création d'une page Web racine avec du style

(Conseil : à faire après avoir réalisé toutes les tâches des autres workspaces)

Objectifs :
  • Réaliser une page Web de A à Z
  • Apprendre à créer un menu de navigation
  • Jouer avec les balises HTML et le style CSS
  • Rajouter du contenu dynamique avec Javascript
Contexte

Un menu de navigation sert à créer des liens d'une page Web p1 vers plusieurs autres pages p2,p3....,pn.

Dans le contexte du projet, le but principal de cette partie est de créer une structure logique de navigation vers toutes les pages Web du projet.
Pour cela vous allez modifier cette page Web appelée accueil.html. Cette page serait la première page racine qu'un utilisateur visiterait si elle et les autres pages constituant le site Web du projet étaient sur Internet.

En utilisant les balises HTML de votre choix et en vous inpirant de toutes les pages HTML du projet, faites en sorte que cette page contienne :
  1. un header contenant le titre du projet
  2. un menu de navigation qui redirige vers tous les supports HTML de chaque workspace :
    • support 1 (1_support.html)
    • support 2 (2_support.html)
    • support ...
  3. un body qui contient deux grandes sections :
    • une première section qui contient la copie de l'intégralité du body de la version initiale de cette page HTML
    • une autre section où chaque membre de l'équipe doit y créer son propre article. Chaque membre de l'équipe devra diviser son propre article en 5 parties :
      1. pour indiquer son nom, prénom et adresse email
      2. pour indiquer les 9 styles CSS différents que vous allez appliquer à votre section
      3. pour indiquer les 4 styles que vous allez incorporer sur l'ensemble de cette page
      4. pour intégrer un contenu dynamique (différents des autres membres de l'équipe) en Javascript à appliquer sur votre section
      5. une dernière partie pour expliquer les difficultés rencontrées dans ce dernier travail à réaliser
  4. un Footer avec un tableau contenant:
    • Nom, prénom et email de chaque membre de l'équipe
Ajouter du style CSS à votre page

C'est le moment de vous amuser avec le CSS et les balises HTML.

Chacun de vous devra au minimum intégrer 13 mises en forme CSS différentes dans cette page (dont 9 pour sa propre section).

Vous avez le droit d'utiliser également des exemples du w3schools pour le CSS ou d'utiliser des boîtes à outils comme W3.CSS ou Bootstrap 4 (très utilisée)

Aidez vous uniquement des exemples du site w3schools pour les balises HTML : w3school

La beauté, le design, l'harmonie des couleurs de la page n'est pas le critère ici... Si certain groupe essaie, ça peut devenir un bonus.
Le but principal est que vous essayez de mettre en forme une page Web.
